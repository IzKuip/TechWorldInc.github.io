Welcome to ArcOS


ArcOS is a super advanced, Electron-based Operating System Environment
that sports all kinds of neat features, such as a working file manager, easy
to use application loader, a large amount of personalization, support for multiple
accounts with each having the option to add a Argon2i encrypted password, and
more. With it being in development since February 17th, 2021, we've done our
best to deliver what we believe to be an amazing experience.
